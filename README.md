## ğŸš€ Tentang Proyek

Proyek ini adalah landing page satu halaman (Single Page Application style) yang dirancang untuk menampilkan profil perusahaan, layanan, dan portofolio proyek secara profesional. Website ini mengadopsi desain modern dengan dukungan Dark Mode dan animasi interaktif.

### Tujuan:
- Meningkatkan kepercayaan klien melalui tampilan profesional.
- Menampilkan portofolio proyek dengan galeri interaktif.
- Memudahkan calon klien untuk menghubungi perusahaan via Email atau WhatsApp.
- Optimasi mesin pencari (SEO) untuk cakupan lokal di Timika dan Papua Tengah.

---

## âœ¨ Fitur Utama

1.  **Desain Responsif**: Tampilan optimal di semua perangkat (Desktop, Tablet, Mobile).
2.  **Dark/Light Mode**: Sakelar mode gelap/terang dengan penyimpanan preferensi di Local Storage.
3.  **Hero Section Dinamis**: Background slider otomatis dengan efek parallax dan fade.
4.  **Lightbox Gallery**: Galeri proyek profesional dengan fitur zoom, navigasi keyboard, dan thumbnail strip.
5.  **Modal Interaktif**:
    - Modal detail proyek.
    - Modal kontak dengan validasi form real-time.
6.  **Integrasi Kontak**: Tombol aksi langsung ke Email dan WhatsApp dengan format pesan otomatis.
7.  **Animasi Halus**: Efek scroll reveal dan transisi halus menggunakan CSS Transition & Intersection Observer API.
8.  **SEO Ready**: Implementasi Meta Tags, Open Graph, Twitter Cards, Schema.org JSON-LD, Sitemap, dan Robots.txt.

## ğŸ” Optimasi SEO

Website ini telah dilengkapi dengan fitur SEO lengkap:

- **Meta Tags**: Deskripsi, kata kunci, dan penulis.
- **Open Graph & Twitter Cards**: Tampilan menarik saat link dibagikan di media sosial.
- **Structured Data (JSON-LD)**: Tipe `LocalBusiness`, `Organization`, dan `BreadcrumbList` untuk membantu Google memahami konten bisnis.
- **Sitemap.xml**: Daftar URL valid untuk diindeks mesin pencari.
- **Robots.txt**: Instruksi izin akses bagi crawler.
- **Geo Meta Tags**: Tag lokasi geografis untuk pencarian lokal (Timika, Papua).
